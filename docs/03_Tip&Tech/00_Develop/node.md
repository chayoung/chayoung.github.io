## Node 설치
``` text
# Windows
npm install -g n

n stable     // 안정 버전 설치  
n latest     // 최신 버전 설치  
n lts        // lts 버전 설치  
n x.x.x      // 특정 버전 설치 ( x.x.x 버전 )  
  
// 이전 버전들 삭제해 주는 명령어  
n prune


# Mac
brew search node

# nvm 설치
arch -arm64 brew install nvm


```
