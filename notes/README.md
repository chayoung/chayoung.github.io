# Notes

이 폴더는 Obsidian과 LLMWiki가 함께 읽는 비공개 메모 공간입니다.

## Purpose

- 초안 작성
- 조사 메모
- 운영 기록
- 발행 전 아이디어 정리

## Publishing Rule

- `notes/` 안의 파일은 Hexo에서 발행하지 않습니다.
- 공개할 글만 `source/_posts/` 로 옮깁니다.

## Suggested Structure

- `notes/inbox/` : 임시 메모
- `notes/research/` : 조사/참고 정리
- `notes/drafts/` : 발행 전 초안
- `notes/ops/` : 운영/장애/설정 기록
